CSS
===

Vertically center a div within another div
------------------------------------------

::

    .parent-element {
        -webkit-transform-style: preserve-3d;
        -moz-transform-style: preserve-3d;
        transform-style: preserve-3d;
    }

    .element {
        position: relative;
        top: 50%;
        transform: translateY(-50%);
    }
